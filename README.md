# مساعد الطالب - الكلية التطبيقية 🎓

موقع ذكاء اصطناعي صوتي كامل لمساعدة طلاب الكلية التطبيقية بجامعة القصيم.

## 🚀 طريقة التشغيل

### 1. الحصول على مفتاح Gemini API (مجاني)
1. اذهب إلى: https://aistudio.google.com/app/apikey
2. سجّل دخول بحساب Google
3. انقر "Create API Key"
4. انسخ المفتاح

### 2. إضافة المفتاح
افتح ملف `app.js` وابحث عن السطر:
```js
const GEMINI_API_KEY = "YOUR_GEMINI_API_KEY";
```
واستبدل `YOUR_GEMINI_API_KEY` بمفتاحك.

### 3. تشغيل الموقع
- افتح `index.html` في **Google Chrome** (مطلوب لدعم التعرف على الصوت)
- اسمح بالوصول إلى المايكروفون عند الطلب
- المايكروفون يعمل تلقائياً بمجرد فتح الصفحة!

## ✨ المميزات
- 🎤 مايكروفون دائم التشغيل (يعيد التشغيل تلقائياً)
- 🔊 ردود صوتية بالعربية
- 🤖 هولوجرام متحرك يتفاعل مع الكلام
- 💬 محادثة نصية وصوتية في آنٍ واحد
- ⚡ أسئلة سريعة بنقرة واحدة
- 🌌 خلفية جسيمات تفاعلية
